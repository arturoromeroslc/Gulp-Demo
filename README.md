## Example codes

### Commands

- install global:
	```$ npm install -g gulp```
- install local:
	```$ npm install --save-dev gulp```
	```$ npm install --save-dev gulp-other-plugins```
- run task:
	```$ gulp task_name```
- run multi tasks:
	```$ gulp task_name other_task```
